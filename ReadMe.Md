# Employment Tracker App

- Description:
    This is going to be an appplication used by individuals looking for a Job. This app will help you keep track of Jobs you have applied to: It will tell you where you filled out the application, keep you updated on your status. This application will also add any interviews to your Google, Icloud, or Outlook calendars.
